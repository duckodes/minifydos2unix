# 行尾符號轉換工具

C# 撰寫的輕量化工具，用於在文字檔中轉換行尾符號格式：  
- LF (Unix/Mac)  
- CRLF (Windows)  

本工具僅進行行尾符號轉換，不修改檔案編碼

## 功能
- LF/CRLF 互轉
- 保留原始檔案編碼 (如 UTF-8、UTF-16)

## 安裝
1. 執行安裝程式並完成設定
2. 安裝完成後，可從桌面捷徑或開始功能表啟動

## 使用方式
1. 開啟 **行尾符號轉換工具**
2. 點選 **選擇檔案**，挑選欲轉換的文字檔
3. 選擇目標行尾格式 (LF 或 CRLF)
4. 即可輸出成目標行尾格式

## 注意事項
- 僅適用於文字檔，請勿用於二進位檔。
- 不更改檔案編碼，只處理行尾符號。
